### **System Overview**
```
React Frontend (5173)
         ↓ HTTP
    FastAPI (8000)
    ↓    ↓    ↓
  DuckDB FAISS Ollama
```

### **Technology Stack**
- **Frontend**: React 18.3, Vite 5.4, TypeScript
- **Backend**: FastAPI 0.139, Uvicorn 0.51, Python 3.13
- **Database**: DuckDB 1.5.4, FAISS
- **AI**: Ollama, qwen2.5:7b, nomic-embed-text
- **Data**: Polars, Parquet, Openpyxl

### **API Endpoints** (Quick Reference)
- `POST /api/v1/upload` - Upload file
- `POST /api/v1/chat` - Chat message
- `GET /api/v1/chart/{id}` - Get charts
- `POST /api/v1/report/generate` - Generate report
- `GET /api/v1/health` - Health check
- Plus 25+ more (see docs)

### **Performance Metrics**
- File upload: 2-5 seconds
- Chat response: 3-5 seconds
- Report generation: 5-15 seconds

### **Data Processing Pipeline**
Excel → Parse → Clean → Normalize → Parquet → DuckDB → RAG

### **AI Chat Pipeline**
Question → LLM Plan → RAG → SQL Gen → Execute → Answer Gen → Chart → Response
